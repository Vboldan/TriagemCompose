# TriagemCompose

Aplicativo Android desenvolvido com Jetpack Compose para triagem de pacientes.

## Funcionalidades
- Cadastro de pacientes
- Classificação por cor (Verde, Amarelo, Vermelho)
- Reclassificação automática após 1 hora
- Persistência local com DataStore
- Interface responsiva com Jetpack Compose

## Como usar
1. Preencha nome, idade e sintomas
2. Escolha a cor de triagem
3. Confirme e visualize os pacientes triados
